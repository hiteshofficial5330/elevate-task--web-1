# Responsive Landing Page - Web Development Internship Task 1

This repository contains the solution for Task 1 of the Web Development Internship from Elevate Labs. The task was to create a clean, simple, and responsive landing page using only HTML and CSS.

[cite_start]The main goal was to build a single-page website featuring a header, a hero section, and a footer[cite: 6]. The key requirement was to ensure the page is fully responsive and looks good on all devices, from mobile phones to desktops.
